# 蓝血精英 - 响应式设计规范

## 断点系统

```css
/* 断点定义 */
--breakpoint-sm: 640px;   /* 大手机 */
--breakpoint-md: 768px;   /* 平板 */
--breakpoint-lg: 1024px;  /* 小桌面 */
--breakpoint-xl: 1280px;  /* 标准桌面 */
--breakpoint-2xl: 1440px; /* 大桌面 */
```

### 断点别名
| 别名 | 范围 | 设备类型 |
|-----|------|---------|
| `mobile` | < 640px | 手机 |
| `tablet` | 640px - 1023px | 平板 |
| `desktop` | 1024px+ | 桌面 |
| `lg-desktop` | 1280px+ | 大桌面 |

## 客户端响应式

### 布局框架

#### 移动端 (< 640px)
```
┌─────────────────────┐
│      顶部栏          │  56px, 固定
├─────────────────────┤
│                     │
│      主内容区        │  100% - 32px边距
│    (底部预留64px)    │
│                     │
├─────────────────────┤
│     底部导航栏       │  64px, 固定
└─────────────────────┘
```

**布局规则**:
- 容器宽度: 100% - 32px (左右各16px)
- 卡片单列布局
- 底部固定导航栏
- 顶部栏简化(隐藏部分导航)

#### 平板端 (640px - 1023px)
```
┌─────────────────────┐
│      顶部栏          │  56px
├─────────────────────┤
│                     │
│   主内容区           │  100% - 48px边距
│  (侧边栏可选展开)     │
│                     │
├─────────────────────┤
│   底部导航栏(可选)   │  可隐藏，用侧边栏替代
└─────────────────────┘
```

**布局规则**:
- 容器宽度: 100% - 48px (左右各24px)
- 卡片2列布局
- 可选择显示底部导航或侧边栏
- 顶部栏显示完整导航

#### 桌面端 (1024px+)
```
┌─────────────────────────────────┐
│          顶部栏                  │ 56px
├─────────────────────────────────┤
│                                 │
│      主内容区 (居中)             │ max-width: 1200px
│                                 │
│                                 │
└─────────────────────────────────┘
```

**布局规则**:
- 最大宽度: 1200px，居中
- 卡片3-4列布局
- 隐藏底部导航
- 完整顶部导航

### 卡片

```css
/* 任务卡片 */
.task-card {
  /* 移动端: 单列 */
  @media (max-width: 639px) {
    width: 100%;
  }

  /* 平板: 2列 */
  @media (min-width: 640px) and (max-width: 1023px) {
    width: calc(50% - 12px);
  }

  /* 桌面: 3列 */
  @media (min-width: 1024px) {
    width: calc(33.333% - 16px);
  }
}

/* 用户卡片 */
.user-card {
  /* 移动端: 2列小卡片 */
  @media (max-width: 639px) {
    width: calc(50% - 8px);
  }

  /* 平板: 3列 */
  @media (min-width: 640px) and (max-width: 1023px) {
    width: calc(33.333% - 16px);
  }

  /* 桌面: 4列 */
  @media (min-width: 1024px) {
    width: calc(25% - 24px);
  }
}
```

.container {

    padding: 0 24px;
  }

  @media (min-width: 1024px) {

## 响应式优先策略

本项目采用**移动端优先**策略：

```css
/* 基础样式 - 移动端 */
.component {
  /* 移动端样式 */
}

/* 平板 */
@media (min-width: 640px) {
  .component {
    /* 平板样式 */
  }
}

/* 桌面 */
@media (min-width: 1024px) {
  .component {
    /* 桌面样式 */
  }
}
```
