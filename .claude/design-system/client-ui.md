# 蓝血精英 - 客户端UI设计

## 页面结构总览

```
Layout (AppShell)
├── TopNav (顶部导航 - 桌面端)
├── MobileTabBar (底部导航 - 移动端)
└── Main Content (主内容区)
    ├── 发现页 (/discover)
    ├── 任务页 (/tasks)
    ├── 课程页 (/courses)
    ├── 岗位页 (/jobs)
    ├── 消息页 (/messages)
    ├── 我的页 (/me)
    └── 任务详情 (/tasks/:id)
    └── 课程详情 (/courses/:id)
    └── 用户主页 (/users/:id)
```

## 8. 移动端适配

### 响应式断点

```
Mobile:   < 640px
Tablet:   640px - 1024px
Desktop:  > 1024px
```

### 移动端特殊处理

```
1. 底部导航
   - 固定在底部
   - 安全区适配 (env(safe-area-inset-bottom))
   
2. 顶部导航
   - 简化：只显示Logo和操作按钮
   - 滚动时隐藏/显示

3. 卡片布局
   - 网格改为单列或双列
   - 增大点击区域

4. 模态框
   - 全屏或底部弹出
   - 下滑关闭手势

5. 表单
   - 输入框高度增大 (56px)
   - 键盘弹出时的布局调整

6. 字体
   - 最小字号不小于14px
   - 行高适当增加
```

### 安全区适配

```css
/* 底部导航 */
.mobile-tab-bar {
  padding-bottom: env(safe-area-inset-bottom, 0);
}

/* 页面内容 */
.page-content {
  padding-bottom: calc(64px + env(safe-area-inset-bottom, 0));
}
```
