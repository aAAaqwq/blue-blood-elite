# 蓝血精英 - 管理端UI设计

## 管理端概览

管理端采用**桌面优先**设计，信息密度高，操作效率优先。布局为固定侧边栏 + 主内容区。

## 布局框架

### 整体结构
```
┌─────────────────────────────────────────────────────────────┐
│  Logo        蓝血精英管理后台           [通知] [头像] [退出]  │  ← 顶部栏 56px
├──────────┬──────────────────────────────────────────────────┤
│          │                                                  │
│  侧边栏   │              主内容区                            │
│  260px   │         (最大宽度1440px, 居中)                    │
│          │                                                  │
│          │                                                  │
│          │                                                  │
└──────────┴──────────────────────────────────────────────────┘
```

### 顶部栏 (56px)
- **背景**: Blue 900 (#0A1628)
- **底部边框**: 0.5px solid Blue 600
- **左侧**: Logo + "管理后台"文字
- **右侧**: 通知图标 + 管理员头像 + 下拉菜单

### 侧边栏 (260px 固定)
- **背景**: Blue 800 (#0F1D32)
- **右边框**: 0.5px solid Blue 600

**导航结构**:
```
仪表盘
├── 数据概览
├── 实时指标

用户管理
├── 用户列表
```

## 交互规范

### 表格行操作
- **悬停**: 整行高亮 + 操作按钮显现
- **点击**: 打开详情抽屉
- **长按/右键**: 快捷菜单

### 批量操作
- **复选框**: 表头全选, 单行独立选择
- **批量工具栏**: 选中时顶部显示
- **操作确认**: 危险操作二次确认

### 数据刷新
- **自动刷新**: 仪表盘每 30 秒
- **手动刷新**: 列表页右上角按钮
- **刷新提示**: Toast 通知

### 加载状态
- **骨架屏**: 首屏加载
- **Spinners**: 局部操作
- **进度条**: 大文件上传/导出

---

*管理端设计强调效率和信息密度，所有操作应在3步内完成。*
