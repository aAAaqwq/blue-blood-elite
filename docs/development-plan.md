## 五、技术决策记录

| 决策 | 选择 | 理由 |
|------|------|------|
| 前端框架 | Next.js 14 | SSR + API Routes一体 |
| 链 | Base (L2) | Gas低 + USDC原生 + Coinbase背书 |
| 钱包 | Privy | 无感上链 + MPC安全 |
| 数据库 | Supabase (PG) | 免费 + 实时 + Row Level Security |
| 合约框架 | Foundry | 编译快 + 测试好 + Solidity原生测试 |
| 部署 | Vercel | 零运维 + 全球CDN |
| MVP不做 | 小程序/App/法币/课程/黑客松 | 严格MVP范围控制 |

## 六、MVP验收Checklist

### 核心流程（必须全部通过）

- [ ] 新用户注册 → 自动创建Privy钱包 → 进入个人资料编辑
- [ ] 编辑个人资料 → 上传头像 + 填写技能标签 → 保存成功
- [ ] 发布任务 → 填写信息 → 存入USDC → 链上Escrow锁定成功
- [ ] 另一用户浏览任务列表 → 筛选/搜索正常
- [ ] 点击任务 → 查看详情 → 链上Escrow状态正确显示
- [ ] VERIFIED用户申请认领 → 发布方审核 → 认领成功 → 合约记录executor
- [ ] 工程师提交交付 → 发布方确认验收 → 合约自动释放USDC
- [ ] 工程师钱包余额更新 → 交易记录显示
- [ ] 任务完成后信誉SBT铸造 → 个人主页显示信誉分

### 非功能性

- [ ] 移动端（iPhone/Android）浏览器正常使用
- [ ] 页面加载 < 3秒
- [ ] 链上交易确认进度有显示
- [ ] BaseScan可查到所有链上交易
- [ ] 错误场景有友好提示（余额不足、网络错误等）
