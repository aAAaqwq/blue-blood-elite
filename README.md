## 核心功能模块

### 智能合约（v2.0 新增）
- 🔗 **Escrow托管合约** — 资金锁定 + 里程碑分批释放 + 多签仲裁
- 🔗 **链上信誉SBT** — 不可转让的信誉Token，新手→可靠→精英→传说
- 🔗 **NFT成就徽章** — 黑客松/任务/收入里程碑自动铸造

### 产品原型 Tab（v3.1）
- 🔍 发现 — 任务发现+社区动态
- ⚡ 成长 — 学习路径+技能认证
- 🏛 任务 — 任务发布/接单+Escrow托管
- 👤 我的 — 个人中心+链上履历

## 技术架构

| 层级 | 技术 |
|------|------|
| 主链 | Base (Ethereum L2) |
| 合约 | Solidity + ERC-5192(SBT) + ERC-721(NFT) |
| 支付 | USDC稳定币 |
| 钱包 | Privy MPC（用户无感上链） |
| 仲裁 | Gnosis Safe多签 |
